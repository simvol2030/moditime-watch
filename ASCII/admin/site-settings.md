## Где используются эти данные

| Данные | Где используется |
|--------|-----------------|
| Название сайта | title страниц, копирайт, мета-теги |
| Логотип header | SiteHeader, CityHeader |
| Логотип footer | SiteFooter, CityFooter |
| Телефон | topbar, header actions, footer, city header/footer |
| Email | topbar, footer, city footer |
| Адрес | footer (секция "Офис") |
| Часы работы | footer (секция "Офис") |
| Соцсети | footer, city footer (telegram) |
| Telegram группа | Telegram CTA секция на главной, city footer |
| Копирайт | footer, city footer |
| Режим телефона | header (direct vs callback) |
