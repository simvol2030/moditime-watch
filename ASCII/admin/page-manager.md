# Админка: Управление страницами

> Центральный экран для просмотра и перехода к редактированию страниц.
> В Moditime Watch используются два типа страниц:
> - **Единичные** (homepage, about, contacts) — нельзя удалить, только редактировать
> - **Масштабируемые** (articles, city articles) — можно создавать/удалять

## Примечания

- Page Manager — **навигационный хаб**, не полноценный CRUD
- Создание/удаление страниц происходит в специализированных разделах (Статьи, Города и т.д.)
- Единичные страницы — фиксированный список, определённый разработчиком
- Клик [✏️] → deeplink в соответствующий раздел админки

| Список городов | ЕСТЬ (раздел "Города") |
