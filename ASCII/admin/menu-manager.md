# Админка: Управление меню

> Сайт имеет несколько независимых меню.
> Каждое меню управляется отдельно.
> Хранилище: таблицы `navigation_items` (header) и `footer_sections` + `footer_links` (footer)

## Список меню

```
┌─ Управление меню ────────────────────────────────────────────────────────────┐
│                                                                               │
│  ┌────────────────────────────────────────────────────────────────────────┐  │
│  │ Название               │ Идентификатор    │ Пунктов │ Действия │      │  │
│  │────────────────────────────────────────────────────────────────────────│  │
│  │ Навигация (desktop)     │ header_desktop   │ 5       │ [✏️]     │      │  │
│  │ Навигация (mobile)      │ header_mobile    │ 5       │ [✏️]     │      │  │
│  │ Footer секции           │ footer           │ 3       │ [✏️]     │      │  │
│  │ Footer правовая         │ footer_legal     │ 2       │ [✏️]     │      │  │
│  │ City навигация          │ city_nav         │ 3       │ [✏️]     │      │  │

```
