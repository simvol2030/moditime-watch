# Админка: Главная страница (Homepage)

> Единичный шаблон — только один экземпляр.
> Каждая секция редактируется отдельной вкладкой/аккордеоном.

---

## Навигация по секциям

```
┌─ Управление главной ─────────────────────────────────────────────────────────┐
│                                                                               │
│  [Hero] [Коллекции] [Бестселлеры] [Сервисы] [Отзывы] [Журнал] [Telegram]    │
│  ─── активная вкладка подчёркнута ───                                        │
│                                                                               │
│  Каждая вкладка — отдельная форма с [Сохранить]                              │
│                                                                               │
└───────────────────────────────────────────────────────────────────────────────┘
```

## Общие действия на каждой вкладке

```
[Сохранить]  [Сбросить к дефолту]
```

- **Сохранить:** применяет изменения мгновенно (на публичном сайте)
- **Сбросить к дефолту:** возвращает значения к начальным (seed data)

## Сценарии использования

```
СЦЕНАРИЙ: Обновление Hero-секции
  1. Админ переходит в Управление главной → вкладка Hero
  2. Меняет заголовок и описание
  3. Загружает новое изображение
  4. Нажимает [Сохранить]
  5. Главная страница мгновенно обновляется

СЦЕНАРИЙ: Добавление коллекции
  1. Админ переходит во вкладку Коллекции
  2. Нажимает [+ Добавить коллекцию]
  3. Заполняет slug, название, описание, загружает фото
  4. Добавляет товары в коллекцию из поиска
  5. Сохраняет коллекцию
  6. Сохраняет порядок (drag-and-drop)
  7. Новая карточка появляется в карусели на главной

СЦЕНАРИЙ: Смена бестселлеров
  1. Вкладка Бестселлеры → переключает на "Ручной" режим
  2. Удаляет старые товары, ищет и добавляет новые
  3. Сохраняет
  4. Карусель обновляется с новыми товарами

СЦЕНАРИЙ: Отключение Telegram-секции
  1. Вкладка Telegram → снять галочку "Показывать секцию"
  2. Сохранить
  3. Секция скрывается на главной

ОШИБКИ:
  - Превышен max длины → подсветка поля, "Максимум {N} символов"
  - Файл > 2MB → "Максимальный размер файла 2MB"
  - Невалидный URL → "Введите корректный URL"
  - Попытка добавить > макс. элементов → кнопка [+ Добавить] неактивна
```
