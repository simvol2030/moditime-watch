## Секция 1: Hero
[ЕСТЬ]

```
┌─ Hero ──────────────────────────────────────────────────────────────────────────┐
│                                                                                  │
│  {tagline}                                                                       │
│                                                                                  │
│  {Заголовок H1}                           ┌──────────────────────┐              │
│                                            │                      │              │
│  {Описание}                                │   {Hero Image}       │              │
│                                            │   520×640px          │              │
│  [{primary CTA}] [{secondary CTA}]        │                      │              │
│                                            │   ┌─ Badge ────────┐│              │
│  ┌─ Stats ──────────────────────┐         │   │ {badge_label}  ││              │
│  │ {value1}  {value2}  {value3} │         │   │ {badge_title}  ││              │
│  │ {label1}  {label2}  {label3} │         │   └────────────────┘│              │
│  └──────────────────────────────┘         └──────────────────────┘              │
│                                                                                  │
│  ┌─ Quick Links ────────────────────────────────────────────────────────────┐   │
│  │  [{chip1}]  [{chip2}]  [{chip3}]  [{chip4}]  [{chip5}]                   │   │
│  └──────────────────────────────────────────────────────────────────────────┘   │
│                                                                                  │
│  ┌─ Brands ─────────────────────────────────────────────────────────────────┐   │
│  │  {Brand1}  •  {Brand2}  •  {Brand3}  •  {Brand4}  •  {Brand5}           │   │
│  └──────────────────────────────────────────────────────────────────────────┘   │
│                                                                                  │
└──────────────────────────────────────────────────────────────────────────────────┘
```

Элементы:
- **Tagline:** текст, max 80 — из БД (home_hero)
- **Заголовок H1:** текст, max 150, serif font — из БД (home_hero)
- **Описание:** текст, max 500 — из БД (home_hero)
- **Primary CTA:** текст + href — из БД (home_hero → primary_cta_text, primary_cta_href)
- **Secondary CTA:** текст + href — из БД (home_hero → secondary_cta_text, secondary_cta_href)
- **Hero Image:** изображение + alt + badge — из БД (home_hero)
- **Image Badge:** label + title (наложение на изображение) — из БД
- **Stats:** массив [{value, label}] — из БД (home_hero → stats_json). Динамические: подсчёт товаров/брендов из products/brands
- **Quick Links:** массив [{text, href, variant}] — из БД (home_hero → quick_links_json)
- **Brands:** массив строк — из БД (home_hero → brands_json)
