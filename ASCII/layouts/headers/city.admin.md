# Админка: Header "City"

> City header наследует глобальные настройки (логотип, телефон) из "Настройки сайта".
> Специфичные для города данные берутся из таблицы `cities`.

## Редактирование

```
┌─ City Header: настройки ─────────────────────────────────────────────────────┐
│                                                                               │
│  Логотип: (→ Настройки сайта)                                                │
│  Телефон: (→ Настройки сайта)                                                │
│                                                                               │
│  ┌─ City Nav Items ───────────────────────────────────────────────────────┐  │
│  │                                                                         │  │
│  │  Управляется через: таблицу city_nav_items                             │  │
│  │  (→ отдельный раздел админки "Города")                                 │  │
│  │                                                                         │  │
│  └─────────────────────────────────────────────────────────────────────────┘  │
│                                                                               │
│  Показывать поиск:        [x] да                                             │
│  Показывать тему:         [x] да                                             │
│                                                                               │
│  [Сохранить]                                                                  │
│                                                                               │
└───────────────────────────────────────────────────────────────────────────────┘
```

## Примечания

- City header минимален — основные данные из глобальных настроек и города
- Бейдж "Часы в {городе}" формируется автоматически из данных города
- Ссылка "Главный каталог" — захардкожена, всегда ведёт на `/`
- Нет бургер-меню — на мобильном просто скрываются nav items
