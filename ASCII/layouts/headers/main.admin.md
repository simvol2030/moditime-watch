## Сценарии использования

```
СЦЕНАРИЙ: Смена текста topbar
  1. Админ переходит в Настройки → Header
  2. Меняет текст бейджа и описания
  3. Нажимает [Сохранить]
  4. Topbar обновился на всех публичных страницах

СЦЕНАРИЙ: Скрытие topbar
  1. Админ снимает галочку "Показывать topbar"
  2. Сохраняет
  3. На всех страницах header начинается сразу с навигации

СЦЕНАРИЙ: Переключение на callback-режим телефона
  1. Админ выбирает "Форма обратного звонка"
  2. Сохраняет
  3. Клик по телефону открывает модалку вместо прямого звонка

СЦЕНАРИЙ: Загрузка логотипа-изображения
  1. Админ выбирает режим "Изображение"
  2. Загружает SVG/PNG файл
  3. Текстовый wordmark заменяется на изображение
```

## Текущее состояние

| Элемент | Статус | Примечание |
|---------|--------|-----------|
| Topbar тексты | ЕСТЬ (хардкод) | Сейчас захардкожены в компоненте |
| Wordmark/Tagline | ЕСТЬ (хардкод) | Не редактируется из админки |
| Навигация | ЕСТЬ (из БД) | Управляется через navigation_items |
| Режим телефона | ЕСТЬ (из БД) | site_config → phone_mode |
| Действия (иконки) | ЕСТЬ (хардкод) | Нельзя скрыть/показать из админки |
| Sticky поведение | ЕСТЬ (хардкод) | Всегда sticky, не настраивается |
