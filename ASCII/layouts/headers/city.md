# Header: City (CityHeader)

Используется в layouts: city (pSEO подсайты)
Компонент: `src/lib/components/layout/CityHeader.svelte`

## Структура

```
┌─ City Header (sticky) ─────────────────────────────────────────────────────────┐
│                                                                                 │
│  [LOGO]  [Часы в {городе}]  ← Главный каталог  {Nav items}   🔍[___] 📞 🌙   │
│  Modit.  ── gold badge ──   ── стрелка-ссылка   из БД                          │
│                                                                                 │
└─────────────────────────────────────────────────────────────────────────────────┘
```

## Элементы

- **Logo:** тот же что и в main header — захардкожен
- **City Badge:** "Часы в {cityNamePrepositional}" — золотой фон, ссылка на `/city/{slug}`
  - Условный: только если данные города есть
- **Ссылка "Назад":** "← Главный каталог" → `/` (стрелка анимируется при hover)
- **City Nav Items:** из БД (`city_nav_items`), горизонтально, скрыты на ≤768px
- **Поиск:** инлайн форма (input 180px desktop / 120px mobile) → `/catalog?q=...`
- **Телефон:** полный номер на desktop, иконка на mobile (из site_config)
- **Тема:** переключатель light/dark

## Поведение

- **Прокрутка:** sticky
- **Мобильная версия (≤768px):**
  - City nav items скрыты
  - Поиск сужается до 120px
  - Телефон → только иконка
  - Нет бургер-меню (упрощённый header)
