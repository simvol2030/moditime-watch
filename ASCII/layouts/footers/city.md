# Footer: City (CityFooter)

Используется в layouts: city (pSEO подсайты)
Компонент: `src/lib/components/layout/CityFooter.svelte`

## Структура

```
┌─ City Footer ──────────────────────────────────────────────────────────────────┐
│                                                                                 │
│  [LOGO]     📞 {телефон*}     📦 Доставка в {город}: {дни} дн., {цена}        │
│  Modit.     ✉️  {email*}                                                       │
│             💬 {telegram*}     Перейти на главный каталог →                     │
│                                                                                 │
│────────────────────────────────────────────────────────────────────────────────│
│                                                                                 │
│  {Копирайт*}                                  {Правовые ссылки}                │
│                                                                                 │
└─────────────────────────────────────────────────────────────────────────────────┘

* = из site_config (глобальные настройки)
```

## Элементы

- **Logo:** тот же wordmark — захардкожен
- **Контакты:** телефон, email, telegram — всё из site_config
- **Доставка в город:** условный блок (если city data есть)
  - Формат: "Доставка в {cityNameAccusative}: {delivery_days} дн., {delivery_price}"
  - Label серый, значения — primary color
- **Ссылка "Каталог":** "Перейти на главный каталог →" → `/` (стрелка анимируется)
- **Копирайт:** глобальный (site_config)
- **Правовые ссылки:** из БД (footer legal links)

## Поведение

- **Упрощённый footer** по сравнению с main — одна строка + юрид. полоса
- **Мобильная версия (≤768px):** вертикальная колонка, центрированный текст
