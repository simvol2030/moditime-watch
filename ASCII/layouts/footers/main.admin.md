> Глобальные элементы (телефон, email, адрес, соцсети) редактируются в "Настройки сайта".
> Секции и ссылки footer управляются здесь.

## Сценарии использования

```
СЦЕНАРИЙ: Добавление новой секции в footer
  1. Админ нажимает [+ Добавить секцию]
  2. Вводит заголовок секции: "Поддержка"
  3. Добавляет ссылки: "FAQ", "Возврат", "Гарантия"
  4. Перетаскивает секцию в нужную позицию
  5. Сохраняет
  6. Новая колонка появляется в footer на всех публичных страницах

СЦЕНАРИЙ: Редактирование ссылки
  1. Админ нажимает [✏️] у ссылки
  2. Меняет название или URL
  3. Сохраняет

СЦЕНАРИЙ: Обновление копирайта
  1. Админ меняет текст копирайта
  2. Сохраняет
  3. Обновляется на всех страницах (год подставляется автоматически)
```

## Текущее состояние

| Элемент | Статус | Примечание |
|---------|--------|-----------|
| Описание бренда | ЕСТЬ (хардкод) | Не редактируется из админки |
| Секции + ссылки | ЕСТЬ (из БД) | footer_sections + footer_links |
| Копирайт | ЕСТЬ (из БД) | site_config → copyright_text |
| Правовые ссылки | ЕСТЬ (из БД) | Секция "Правовая информация" |
| Адрес/часы работы | ЕСТЬ (из БД) | site_config |
