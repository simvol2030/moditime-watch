# Footer: Main (SiteFooter)

Используется в layouts: public
Компонент: `src/lib/components/layout/SiteFooter.svelte`

## Структура

```
┌─ Footer ───────────────────────────────────────────────────────────────────────┐
│                                                                                 │
│  [LOGO]              {Секция 1}         {Секция 2}        {Секция N}           │
│  Moditimewatch       ── заголовок ──    ── заголовок ──   ── заголовок ──      │
│                                                                                 │
│  {Описание сервиса}  Ссылка 1           Ссылка 1          Ссылка 1            │
│                      Ссылка 2           Ссылка 2          Ссылка 2            │
│  📞 {телефон*}       Ссылка 3           Ссылка 3          ...                 │
│  ✉️  {email*}                                                                  │
│                                                                                 │
│  ┌─ Секция "Офис" (особая) ────────────────────────────────────────────────┐   │
│  │  {адрес*}                                                                │   │
│  │  {часы работы*}                                                          │   │
│  │  Посещение по записи                                                     │   │
│  └──────────────────────────────────────────────────────────────────────────┘   │
│                                                                                 │
│────────────────────────────────────────────────────────────────────────────────│
│                                                                                 │
│  {Правовая информация — ссылки}                         {Копирайт*}            │
│  Политика конф. | Оферта | ...                          © 2024 Moditimewatch   │
│                                                                                 │
└─────────────────────────────────────────────────────────────────────────────────┘

* = из site_config (глобальные настройки)
```

## Элементы

### Бренд-колонка (первая)
- **Logo:** тот же wordmark — захардкожен
- **Описание:** "Сервис доставки оригинальных часов из-за рубежа..." — захардкожен
- **Телефон:** глобальный (site_config → `contact_phone`), ссылка `tel:`
- **Email:** глобальный (site_config → `contact_email`), ссылка `mailto:`

### Секции контента
- **Источник:** БД → `footer_sections` + `footer_links`
- **Responsive grid:** `repeat(auto-fit, minmax(220px, 1fr))`
- **Заголовки:** uppercase, accent font, letter-spacing 0.25em
- **Особые секции:**
  - "Офис" → показывает адрес + часы работы вместо ссылок
  - "Правовая информация" → переносится в нижнюю полосу

### Нижняя полоса
- **Копирайт:** глобальный (site_config → `copyright_text`), автоподстановка года
- **Правовые ссылки:** горизонтально, из секции "Правовая информация"

## Поведение

- **Фон:** светло-серый (#f5f7fa), тёмная тема: ink color
- **Мобильная версия (≤768px):**
  - Центрированный текст
  - Полноширинные секции
  - Вертикальные правовые ссылки
