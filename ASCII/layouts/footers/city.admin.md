# Админка: Footer "City"

> City footer наследует глобальные настройки (контакты, копирайт) из "Настройки сайта".
> Данные доставки берутся из таблицы `cities`.

## Редактирование

```
┌─ City Footer: настройки ─────────────────────────────────────────────────────┐
│                                                                               │
│  Контакты: (→ Настройки сайта: телефон, email, telegram)                     │
│  Копирайт: (→ Настройки сайта)                                               │
│  Правовые ссылки: (→ Footer Main → секция "Правовая информация")             │
│                                                                               │
│  Данные доставки: (→ управление в "Города" → каждый город)                   │
│    delivery_days, delivery_price, city_name_accusative                        │
│                                                                               │
│  [Сохранить]                                                                  │
│                                                                               │
└───────────────────────────────────────────────────────────────────────────────┘
```

## Примечания

- City footer полностью наследует данные из глобальных настроек и городов
- Собственных настраиваемых элементов почти нет
- Ссылка "Главный каталог" — захардкожена
