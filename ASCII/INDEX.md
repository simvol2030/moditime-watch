## Приоритеты реализации (рекомендуемый порядок)

1. **homepage.admin.md** — управление главной (самый востребованный)
   - Вкладки Hero, Коллекции, Бестселлеры нужны первыми
   - Eyebrow/Title всех секций → из хардкода в БД
2. **site-settings.md** — глобальные настройки
   - Контакты уже в БД, нужен UI
   - Логотип → из хардкода в БД
3. **menu-manager.md** — управление меню
   - Данные уже в БД, нужен UI с drag-and-drop
4. **page-manager.md** — навигационный хаб
   - Объединяет все разрозненные разделы в одном месте
5. **main.admin.md / footers/main.admin.md** — настройки layout
   - Topbar тексты, footer описание → из хардкода в БД
